# RTSL Specification

### Keywords

#### Modules
`import`, `export` `` 
#### Types: 
`struct`, `using`
#### Type and value: 
`const`, `auto`, `void`
#### Control flow: 
`if`, `else`, `while`, `do`, `for`, `return`
#### Varying qualifiers: 
`clip`, `smooth`, `flat`
#### Built-in boolean literals: 
`true`, `false`
#### Resource access qualifiers:
`readonly`, `writeonly`
#### Special
`uniform`, `varying`, `namespace`, `entry`


### Settings
Every Compiler has to implement a few different shader types.
